# FINAL-PROJECT

Small Business Inventory Manager — Streamlit app with JSON-backed storage.

- Run: `streamlit run app.py`
- Data files: `inventory.json`, `sales.json`, `flags.json`, `users.json`
- Service layer: `service.py` handles validation and persistence via `data_access.py`.
